🔗 Åpne GitHub-siden

🚀 Slik laster du opp endringer (3 steg)

1

Legg til alle endrede filer:

git add -A
2

Lagre med en beskrivelse (endre teksten etter -m):

git commit -m "Oppdatert ordliste"
3

Last opp til GitHub:

git push
💡 Tips: Åpne Terminal (PowerShell) i riktig mappe: høyreklikk i Ordliste-mappen i Filutforsker → "Åpne i Terminal"

💻 Åpne terminal i riktig mappe

→

Alternativt, skriv dette i PowerShell for å gå til riktig mappe:

cd "$env:USERPROFILE\Documents\Ordliste"

Sjekk status:
git status

git log --oneline
